Excel-Controlled Arduino & SCPI over TCP/IP (No USB)

Welcome to ExcelControlSolutions — your home for automation with Excel VBA and TCP/IP. This unique toolkit empowers you to control Arduino devices and SCPI instruments directly over your LAN using the native Windows Sockets API (WSA) — no serial ports, no ActiveX controls.

🌐 Key Features

  • 🧠 Native WSA TCP client built in pure Excel VBA
  • 🔌 Control Arduino with W5100/W5500 over Ethernet
  • 📟 Send SCPI commands to lab instruments via LAN
  • 📈 Real-time data logging and automated triggers
  • 🛠 No need for mswinsck.ocx or serial drivers

📦 What's Included

  • Excel workbook with complete WSA TCP module
  • Arduino SCPI command Library for TCP server communication
  • SCPI command templates for Ethernet instruments
  • Excel Workbook with complete logger and event macros for HIOKI IM3570
  • Step-by-step PDF setup guide

🖥 Requirements

  • Windows PC with Excel 2013 or newer
  • Internet or LAN-enabled instruments/boards
  • Arduino with Ethernet (W5100/W5500) or SCPI device
  • Basic understanding of Excel macros (VBA)
